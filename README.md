# app1

A simple greeting app that says hello to you.

## Requirements

- Python 3.13+
- [uv](https://github.com/astral-sh/uv) (recommended)

## Setup

```bash
uv venv
source .venv/bin/activate
```

## How to Play

1. Run the app:
   ```bash
   python main.py
   ```

2. When prompted, enter your name:
   ```
   Enter your name:
   ```

3. The app will greet you:
   ```
   Hello [your name]!
   ```
